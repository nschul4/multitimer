# a website made using angularjs

This site is live at: https://nschul4.github.io/multitimer/

### initialize the project
```
rm -rf node_modules package.js package-lock.js
npm init --yes
npm install angular
```

### install angular
I followed instructions in
[node_modules/angular/README.md#Install](node_modules/angular/README.md#Install).

Quote:
> Then add a `<script>` to your `index.html`:
>
> ```html
> <script src="/node_modules/angular/angular.js"></script>
> ```
